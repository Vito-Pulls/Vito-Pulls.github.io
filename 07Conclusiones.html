Conclusiones

8.Conclusiones

En este proyecto se ha investigado la integración de tecnologías modernas, como sensores de radiación UV, relojes inteligentes y APIs externas para desarrollar una aplicación teórica que calcule y monitoree la síntesis de vitamina D a través de la exposición solar y estilos de vida saludables, promoviendo una exposición saludable a la radiación UV.

Hemos desarrollado un modelo teórico que es totalmente funcional sin el reloj, y otro modelo que, con ajustes adicionales técnicos para implementar un sensor de luz en un reloj inteligente, sería viable para desarrollar la aplicación. A pesar de no haber desarrollado la aplicación funcional debido a la falta de conocimientos técnicos, el estudio ha demostrado que es técnicamente posible integrar sensores de radiación UV en un reloj inteligente y que este se comunique con el backend para realizar cálculos y enviar los resultados al frontend.

A nivel personal, este proyecto ha sido enriquecedor, permitiéndonos expandir nuestros conocimientos básicos en tecnologías de aplicaciones web. La investigación nos ha proporcionado una visión más amplia sobre el funcionamiento, la arquitectura de las aplicaciones web y cómo se podrían utilizar herramientas como las APIs.

Aunque no se logró desarrollar la aplicación, este trabajo ha sido clave para futuras investigaciones sobre la optimización de la síntesis de vitamina D.
